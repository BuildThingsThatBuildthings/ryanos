Quick Start Guide

1. Authentication

All API endpoints (except authentication and health) require a valid JWT token:

# Register a new user curl -X POST https://api.fitnessapp.com/auth/register \ -H "Content-Type: application/json" \ -d '{ "name": "John Doe", "email": "john@example.com", "password": "securePassword123", "confirmPassword": "securePassword123" }' # Login to get access token curl -X POST https://api.fitnessapp.com/auth/login \ -H "Content-Type: application/json" \ -d '{ "email": "john@example.com", "password": "securePassword123" }'

2. Create a Workout

curl -X POST https://api.fitnessapp.com/workouts \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "name": "Push Day", "description": "Chest, shoulders, and triceps", "date": "2024-01-15" }'

3. Add Sets to Workout

curl -X POST https://api.fitnessapp.com/workouts/workout_123/sets \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "exerciseId": "exercise_bench_press", "setNumber": 1, "reps": 8, "weight": 135, "rpe": 7 }'

4. Voice Workout Logging

# Create voice session curl -X POST https://api.fitnessapp.com/voice/sessions \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{"workoutId": "workout_123"}' # Process voice transcript curl -X POST https://api.fitnessapp.com/voice/extract-workout \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "transcript": "I just completed 3 sets of bench press with 135 pounds for 8 reps each" }'

5. Generate AI Workout

curl -X POST https://api.fitnessapp.com/workouts/generate \ -H "Authorization: Bearer YOUR_TOKEN_HERE" \ -H "Content-Type: application/json" \ -d '{ "prompt": "Create a 45-minute upper body strength workout for intermediate level", "preferences": { "duration": 45, "equipment": ["barbell", "dumbbell"], "targetMuscles": ["chest", "shoulders", "triceps"], "level": "intermediate" } }'

Key Endpoints

POST /auth/login

Authenticate user and receive JWT token

GET /workouts

Get user's workouts with pagination and filtering

POST /workouts

Create a new workout

GET /exercises

Browse exercise library with advanced filtering

POST /voice/sessions

Start a voice logging session

POST /workouts/generate

Generate AI-powered workout
